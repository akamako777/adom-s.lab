# 🛠️ Adom's Lab Tools

ブラウザだけで動作する、インストール不要の高機能Webツール集です。
サーバーにデータを送信しない「完全ローカル処理」により、プライバシーとセキュリティを確保しています。

## 🚀 アプリ一覧 & デモ (Live Demo)

各ツールの動作デモと詳細な使い方は、公式ブログをご覧ください。

**👉 公式ブログ: [Adom's Lab Tools](https://adom-s-lab-tools.blogspot.com/)**

### 収録ツール
1. **Omni Image Tools**: 画像の一括リサイズ・WebP変換・透かし追加
2. **Pro Screen Recorder (Aurora)**: インストール不要の画面録画・カメラ録画（ロゴなし）
3. **PDF All-in-One Pro**: PDFの結合・分割・画像化
4. **AI Infinite Kanji Drill**: Gemini APIを活用した漢字ドリル生成
5. **Magic Circle Generator**: 3D魔法陣生成シミュレーター

## 🛡️ プライバシーポリシー
このリポジトリに含まれるツールは、ユーザーのデバイス内（クライアントサイド）ですべての処理を行います。
画像、動画、音声、PDFなどのデータが外部サーバーに送信されることはありません。
