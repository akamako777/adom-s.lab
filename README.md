# 🛠️ Adom's Lab Tools

**Privacy-First & Local-First.**
ブラウザだけで動作するインストール不要のWebツール、および教育現場の業務効率化を支援するGoogle Apps Script (GAS) コレクションです。

ユーザーのプライバシーを最優先し、外部サーバーへのデータ送信を行わない（またはユーザー自身のGoogle Drive内で完結する）設計を徹底しています。

## 🛡️ プライバシーポリシー (Privacy Policy)
本リポジトリのツールは、**「あなたのデータを、あなたの手元で」**処理することを原則としています。

1.  **Webツール (HTML/JS):**
    * すべての処理はユーザーのデバイス内（ブラウザ）で行われます。画像、動画、PDFなどのデータが開発者のサーバーに送信されることはありません。
2.  **GASツール (Google Apps Script):**
    * スクリプトは**実行者自身のGoogleアカウントおよびGoogle Drive内**でのみ動作します。
    * 収集されたアンケートデータや個人情報が、開発者（Adom's Lab）や第三者に送信されることは一切ありません。
